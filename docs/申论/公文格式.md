## 一、公告

<div class="shenlun-auto">
<textarea style="display:none">
{
    title: "<span style='color:red;font-weight:bold'>标题：</span>关于+事由+公告",
    content: [
        { type: "text", text: "称谓（可不写）" },
        { type: "para", text: "<span style='color:red;font-weight:bold'>开头：</span>事由、背景，发布公告的根据、目的、意义。" },
        { type: "para", text: "<span style='color:red;font-weight:bold'>主体：</span>根据材料交代公告内容，简单明了，条理清晰，把握重点，用书面语即可。" },
        { type: "para", text: "<span style='color:red;font-weight:bold'>结尾：</span>可适当提出展望、要求、常以“特此公告”等习惯于结尾。" }
    ],
    footer: ["发文机关 ", "xxxx年xx月xx日"]
}
</textarea>
</div>

## 二、通知

<div class="shenlun-auto">
<textarea style="display:none">
{
    title: "<span style='color:red;font-weight:bold'>标题：</span>关于+事由+通知",
    content: [
        { type: "text", text: "<span style='color:red;font-weight:bold'>称谓：</span>主送机关或称呼" },
        { type: "para", text: "<span style='color:red;font-weight:bold'>开头：</span>事由、发文背景、依据、目的、意义等。" },
        { type: "para", text: "<span style='color:red;font-weight:bold'>主体：</span>1.交待通知事项。总分形式分条叙述指示、工作安排、方法措施步骤等（有些通知需要点名时间、地点、人物等）。2.说明执行的要求。" },
        { type: "para", text: "<span style='color:red;font-weight:bold'>结尾：</span>特此通知/以上通知，望认真执行。" }
    ],
    footer: ["发文机关 ", "xxxx年xx月xx日"]
}
</textarea>
</div>

## 九、主要内容类
> 当题干中出现“提纲”、“内容要点”、“主要内容”、“情况介绍”等关键词，或出现“不考虑行文格式”、“不必写成文章”等，即为主要内容类
>
> “xx提纲”可写标题，“主要内容”一般不写标题

<div class="shenlun-auto">
<textarea style="display:none">
{
    title: "<span style='color:red;font-weight:bold'>标题：</span>关于+事由+纲要",
    content: [
        { type: "para", text: "<span style='color:red;font-weight:bold'>开头：</span>发文缘由、背景、现状、原因、意义等。" },
        { type: "para", text: "<span style='color:red;font-weight:bold'>主体：</span>写基本情况、经验体会、问题教训、未来打算等。" },
        { type: "para", text: "<span style='color:red;font-weight:bold'>结尾：</span>简要概括上文，也可以根据实际情况不写结尾。" }
    ],
    footer: []
}
</textarea>
</div>
