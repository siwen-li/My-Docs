# 基础知识

## 统计术语

*   **倍数**：$倍数 = 增长率 + 1$
*   **翻番**：翻一番为原来的 2 倍，翻两番为原来的 4 倍。**翻 n 番为原来的 $2^n$ 倍**。
*   **成数**：一成相当于总量的 10%，几成相当于总量的百分之几十。
*   **同比**：与历史同期相比较。
    *   【例】2015年7月 vs 2014年7月；2015年1-3月 vs 2014年1-3月。
*   **环比**：包括日/周/月/年环比（与上一个统计周期相比）。
    *   【例】2015年6月 vs 2015年5月。
*   **顺差**：出口 > 进口。 **出口 - 进口 = 顺差**
*   **逆差**：出口 < 进口。 **进口 - 出口 = 逆差** （进出口 = 进口 + 出口，要看清楚！）
*   **产业增加值**：
    *   该行业在周期内（一般以年计）比上个周期的增长值。
    *   国内生产总值（GDP）等于三大产业增加值之和。
    *   **注意：增加值不是增长量！**
*   **人口自然增长率**：= $人口出生率 - 人口死亡率$。**人口自然增长率的单位是千分数 ‰**。

## 其他知识

*   **特定时期表述**：
    *   【新中国成立初】指 1949 年之后的几年。
    *   【改革开放】指 1978 年以后至今。
    *   【十五计划时期】指 2001-2005 年。
    *   【十四五计划时期】指 2021-2025 年 ($4\times5+1=21$ 表示从 21 年开始)。
*   **恩格尔系数**：食品支出总额占家庭或个人消费支出总额的百分比。**越低，生活水平越高**。
*   **基尼系数**：用来衡量一个国家或者地区收入差距的常用指标。**越小，收入差距越小** (0~1之间)。

## 做题原则

1.  **分析问题** $\rightarrow$ **选取关键字** $\rightarrow$ **圈出数据** $\rightarrow$ **列式计算**
2.  **时间**：注意求的是基期还是现期。
3.  **数据**：具体数据略读，重点标注数据前的关键词。
4.  **单位**：优先找不同的单位。
5.  **图例**：饼状图画图规律是从 12 点钟方向开始。
6.  **解题思路**：
    *   1. 找好每一部分的比例关系。
    *   2. 关注特殊的比例：$25\%、50\%$。
