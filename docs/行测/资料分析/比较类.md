# 比较类

## 比值类
*   **趋势比较法**：借用比重趋势解题思路，根据分子分母增速大小判断分数大小

    *   分子增速（倍数）大于分母，则分数变大
    *   分子增速（倍数）小于分母，则分数变小


*   通分法：可将要比较的两个分数的分子或分母换算成同样大小


*   “王者荣耀”法：

    *   利用盐水思想，将分子分母同时拆分，可以判断数字是否大于或小于某个数字




## 增量比较
*   $X = B - A = B \times (1 - \frac{1}{1+R})$。
    *   **B 越大，R 越大，则增量 X 越大**。
    *   B 越大，则左边乘数越大；R 越大，则减掉的部分就越小，则右边乘数越大。
*   **$X = AR = \frac{B}{1+R} \times R$**。
    *   **大大则大，一大一小看倍数**。
   
    *这两句一般在增速为正 (R > 0) 时使用*。

*   **增长率超过 10% 的比较有两种方法**：
    1.  $R = \frac{B-A}{A} = \frac{X}{A} > 10\%$
    2.  $A + 0.1A < B$

## 图表查找（需要注意的坑）
1.  **注意起始、结束年份、月份** (**重中之重**)。
2.  **注意“合计”、“总计”行，以免数错**。
3.  **注意第一年的增量**。
4.  **注意单位**。


## 替代比较

*   用 **末期/基期** 代替 **年均增长率** 比较；
*   用 **倍数（B/A）** 代替 **增速（B/A—1）** 比较；
*   用 **部分A/部分非A** 代替 **部分A/整体比较**；（A/非A  代替  A/整体  或  A/整体  代替  A/非A）
*   用 **X/B** 代替 **X/A** 比较。
