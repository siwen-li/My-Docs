# 比较类

## 比值类（双线法）
*   **分数 / 坐标大小**：适用干增速的大小比较，基期变大，增量变小。
*   **双线法**：
    *   在图表中，一条线代表总量（如收入），一条线代表增速。
    *   $X = B - A = B \times (1 - \frac{1}{1+R})$。
    *   **B 越大，R 越大，则增量 X 越大**。
    *   **B 越大，则左边乘数越大；R 越大，则减掉的部分就越小，则右边乘数越大**。

    > **【例】(2021 年浙江)**
    > 2014~2019 年全球卫星产业收入增长最快的年份是：
    > **【解】** 基期 A 在变大，而增量 X 在变小，$\therefore$ 2014 年增速 $X/A$ 最大。

## 增量比较
*   **$X = AR = \frac{B}{1+R} \times R$**。
*   **大大则大，一大一小看倍数**。
    *   B 越大，则左边乘数越大。
    *   R 越大，则减掉的部分就越小，则右边乘数越大。
    *   我是你的 n 倍，你的 R 是我的 n 倍以上，我们的 X 才能相等。
*   **这两句一般在增速为正 (R > 0) 时使用**。
*   **增长率超过 10% 的比较有两种方法**：
    1.  $R = \frac{B-A}{A} = \frac{X}{A} > 10\%$
    2.  $A + 0.1A < B$

## 图表查找（需要注意的坑）

2.  **注意“合计”、“总计”行，以免数错**。
3.  **注意第一年的增量**。
4.  **注意单位**。
