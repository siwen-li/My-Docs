## 词语的范围
!!! tip ""
    1. **全同**：A 与 B 是同一个事物
    1. **并列**：A 与 B 完全不同，且处于同一层级
        1. **矛盾关系**：只有AB处于同一层级，无第三者
        1. **反对关系**：除了 AB 处于同一层级，至少还有一个第三者
    1. **包含**：A 的范围完全包含 B 的范围 
    1. **交叉**：有的 A 是 B，有的 A 不是 B；有的 B 是 A，有的 B 不是 A
  

### 并列一矛盾关系
常见的矛盾关系：

1. **P-非P**： 合法-不合法；盈利-非盈利；成年人-未成年人
2. **单字**：有-无；动-静；男-女；生-死；开-关；中-外；曲-直；阴-阳

```
实词-虚词；		硬件-软件；		奇数-偶数；
声母-韵母；		白天-黑夜；		唯物-唯心；
麒-麟；		  凤-凰；		  鸳-鸯；		左手-右手；
杂交-纯种；		       X染色体-Y染色体；		急性中毒-慢性中毒；
常绿植物-落叶植物；		连续变量-离散变量；      幸存者-遇难者；
		

```

### 并列一反对关系

> *有第三者*，如：黑-白

二级辨析

1. 排序
	- 是否紧相邻
	- 相对性
	- 变化性
2. 功能
	- 古今、敬辞谦辞、具体抽象

3. **拆分思维-构词结构**
	- 偏正
		- 功能修饰
		- 原理修饰
		- 属性修饰
		- 材料修饰
		- 动植物修饰 
	- 并列
	- 主谓宾
	- 动宾

### 包含一种属关系

> 种属关系造句：**A 是 B（的一种/一个）**



种属二级辨析：

1. **泛指与特指**
    - 种属 **泛指** 造句：A 是 B 的一种；*A 是一种 B*。 喜剧电影：电影
    - 种属 **特指** 造句：A 是 B 的一个；*A 是一个 B*。 《热烈》：电影
2. **反向种属** 
	- *A 不属于 B*
3. 其他：层级细分、拆分思维-构词结构


## 词语的对应 

> 没有思路就造句
> 
> 两词词性做辅助

### 原材料-成品对应

> 词性通常为 **名词：名词**  
> 
> 造句：**（用）A 做 B**

```
黄豆：豆腐
葡萄：发酵：葡萄酒
布料：衣服  
火药：爆竹
玉石：雕刻：玉玺
```

二级辨析：

- **化学变化/物理变化**
	- 化学工艺：发酵 燃烧 氧化 冶炼 高温变质 分解出新物质 等
	- 物理工艺：研磨 压榨 组装 切割  雕刻  雾化  气化  凝结等
- **一种原材料/多种原材料**
- **直接原材料/间接原材料**



### 功能对应
> 词性通常为 **名词：动词/动宾短语**
> 
> 造句：**A 的作用/功能是 B 或者用 AB**


二级辨析：

- **主要功能/次要功能**
```
碘伏：消毒
针灸：疏通经脉
酒精：降温
血液：运输氧气
```


### 属性对应

> 词性通常为 **名词：形容词**
>
> 造句：**A 是 B 的或者 A 的特点是 B**

```
盐：咸		花椒：麻
花：香		饮料：冷
```


二级辨析：

- **必然/可能**
- 注意区分：属性对应与功能对应



### 多个动作排顺序

> 词性特征：**一组动词或者带动词的短语**

```
设计：发放：问卷
制定：执行：政策
```

对应关系：*先排顺序，再看动作的主体是否相同*



### 因果对应

> 造句：**A 是 B 的原因；A 导致 B； B 是 A 引起的**

```
物体振动：自然声源
强光照射：视力损伤
```

二级辨析：

- **结果是好是坏**




### 方式-目的对应

> 造句：**A 是为了 B**

```
削足：适履
刻舟：求剑
```

*预告、预排、预警、谈判等一般都有事先预定的目的*

二级辨析：

- **方式是好是坏**
- **与题干词义是否相近**
- **目的是否达成**

### 必要条件对应

> A 是 B 的必要条件
> 
> 造句：**没有 A，就没有 B**

```
清晨：晨曦
电：电扇
传播介质：自然声源
```

## 词语的语法 
造句的简单原则：

1. 能不加字就不加字
2. 能少加字就少加字
3. 优选相同句式

### 主谓

> **名词+动词** （食物腐烂 花红 柳绿）


### 动宾：动词+名词

例：锤炼本领 安营 扎寨

### 主谓宾：名词+动词+名词

例： 花匠修剪花草 八仙过海


### 偏正：修饰词+中心词

例：生活资料 葡萄酒 清风 朗月

自由飞翔 精打 细算

## 记忆

!!! tip ""

	全同关系找一样，必须一点不能差；对应关亲大杂烩，常识知识要具备

	并列关系看范畴，**矛盾反对** 要分清；选项逻辑都一致，二级辨析帮助你

	交叉关系很隐蔽，**包含范围** 有重叠；古今中外别忘记，自他雅俗也可以

	属性关系看词性，名词形容相对应；语义关系很重要，言语基础要打牢

	因果关系看逻辑，**必然或然** 要分清；语法关系造句子，**主动被动** 看清楚

	包容关系分两种，**种属组成** 各不同
