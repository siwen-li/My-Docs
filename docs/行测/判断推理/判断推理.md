# 6. 分析类题目与相关知识点

## 真假分析

### 矛盾法
1.  **找矛盾**：找到存在矛盾关系，必然一真一假。
2.  **看其余**：结合真假数量确定矛盾以外的真假情况。
    *   **1、只有一真，其余均为假。**
    *   **2、只有一假，其余均为真。**
3.  **矛盾关系**：
    *   A和非A
    *   A或B 和 非A且非B
    *   A且B 和 非A或非B
    *   所有和有些不，可能和必然不
    *   $A \rightarrow B$ 和 $A且非B$

> **【例】** 小陈、小李、小刘3人...
> *   小陈：小李多获奖 (陈且李)
> *   小李：小李至多有一人获奖 (非陈或非李)
> *   小刘：陈且李不获奖 (非陈且非李)
> *   ...
> *   **结果发现，只有1人预测正确。**
> *   **【解】** 第一、二句矛盾，二者必有一真一假，故三四五句全为假。

### 假设法
*   假设条件，根据假设是否矛盾来解题。
*   **【例】** 甲：乙被录；乙：丙被录；丙：甲或乙被录；丁：乙或丙被录...发现只有一句回答符合事实。
*   **【解】** 假设甲为真，则丙、丁也为真，故甲为假。假设乙为真，则丁也为真，故乙为假...

### 代入法
*   代入选项判断命题真假。
*   **【例】** 甲：“它是月季”；乙：“它不是玫瑰”...“你们三人中，只有一个人两次都猜对了，一个人猜对了一次，还有一个全猜错了”。
*   **【解】** 将选项代入判断。

### 两真两假
*   **先找矛盾，再假设。**
*   **命题的真假判断：对于原命题 $A \rightarrow B$：只有 $A且-B$ 时命题才为假。**

## 范畴分析
*   **定义**：所有、有些、特指的相关考点。
*   **一分为二画图法**：找到题干中“所有A都是B”的描述，对B和非B做分类。**所有画实圈，有些不可画圈，不能确定的可以画在线上**。

## 日常分析
*   **代入法**：就是把选项代入题目，依次验证和排除。
*   **赋值法**：对于涉及到大小比较的题目，可以考虑使用赋值法。
*   **画表法**：将已知信息填入图表，更直观进行逻辑分析。

# 7. 判断推理之定义判断

## 解题思路
*   **重在理解，可以联系实际，找到现实所指。**

## 选项设置
*   **看提问**：提问分清“属于”/“不属于”。**(最大陷阱)**
*   **看题干**：识别有效信息——找准关键词、关键句。
*   **看选项**：不要因未体现某一特点，排除某一选项，要关注 **“被定义词”中不包含的信息**。

## 解题技巧
*   **技巧审题**：问啥看啥。
*   **同构选项**：
    *   什么是同构选项？意思相近、结构相似的选项。
    *   怎么用同构选项？同构选项出现，同时排除。
*   **对比枚举型**：选项一一对应。
*   **专业名词/法律术语/医学术语**：需建立在道德基础上，符合“常理”。
*   **涉及时间前后**：需注意选项时间点设置。
*   **涉及变化**：选项设置应有“以前情况”和“现在情况”对比。

## 关键词查找技巧
1.  题目**既有定义名词的本质解释又存在延伸解释时**，抓手常存在于延伸解释中。
2.  定义中有逻辑关联**转折**时，尤其是形成**对比色彩**，一定是抓手。
3.  定义中当有**必要性条件型语句**出现时，该语句很可能成为抓手。
4.  定义中当有**否定性语句**出现时，该语句很可能成为抓手。
5.  定义解释对某一概念进行枚举时，**有限枚举可成抓手**，无限枚举不成抓手。

# 8. 判断推理之类比推理

## 做题思想
*   **大众思维**、**简单思维**。
*   **造句子**：选择相似点和题干最多的。**以我为主：设想更合适的答案是什么**。
*   **一级辨析选不出才考虑二级辨析**。

## 类比3难点
1.  **种属**：A是B的一种。
2.  **组成**：A是B的一个组成部分。
3.  **矛盾**：非A即B。
4.  **特指**：一个。 **泛指**：一类。

## 逻辑关系
*   **圈圈关系**
    *   **全同**：A=B (金陵=南京)
    *   **并列**：
        *   矛盾 (生:死)
        *   反对 (黑:白)
    *   **包容**：
        *   种属 (苹果:水果)
        *   组成 (轮胎:汽车)
    *   **交叉**：从不同角度描述同一事物 (青年人:公务员)

*   **对应关系**
    *   **逻辑性**：
        *   因果关系 (乱砍滥伐:水土流失)
        *   推出关系：
            *   充分条件 (喝水:解渴)
            *   必要条件 (旅行:交通)
    *   **日常性**：
        *   功能 (主/次) (灯:照明 / 灯:装饰)
        *   时间顺序 (一看时间先后，二看主体是否一致，三看主体数量)
        *   方式与目的 (一看通过某种方式能否达到某种目的，二看主体是否一致)
    *   **理论性**：
        *   物理变化 (打磨/捆扎/雕琢) vs 化学变化 (发酵/酿造/冶炼) **区分：有无新物质生成**

## 语文关系
*   **语义关系**
    *   **近义/反义** $\rightarrow$ 二级辨析：感情色彩(褒/贬/中)、程度(好感/喜欢/热爱)
    *   **比喻义/象征义** $\rightarrow$ 二级辨析：词性(名词/动词/形容词)
*   **拆词法**
    *   ① 成语被拆分 (南征:北战)
    *   ② 词语间无明显逻辑关系 (威败:呼吸)
    *   ③ 相同单字反复出现 (寒:寒冷:寒舍)

## 语法结构
*   **主谓** (n.+v. 学生:学习)
*   **动宾** (v.+n. 学习:功课)
*   **主宾** (两词中间+“v.” 学生:功课 $\rightarrow$ 学生“学习”功课) **多作为二级辨析考察**
*   **偏正** (两词中间+“的/地” 漂亮:上衣 $\rightarrow$ 漂亮的衣服)

# 9. 判断推理之图形推理

## 图推十五字口诀
**风臣氏整系，点线面角素，直接想位移。**

### 一、定性分析 (风臣氏整系)
*   **口诀：风臣氏整系 —— 什么时候考虑定性分析？**
    1.  图形比较随意、具有生活化的特点 (如：阴影、简笔画等)。
    2.  图形比较凌乱、不协调的图形，代表考察性质的可能性非常大。
*   **考点：**
    1.  **曲直性**：存在纯曲线时可以考虑。
    2.  **对称性**：出现对称提示时可考虑。
    3.  **相同相似图形**：存在相同元素、非常凌乱时可以考虑。
    4.  **整体部分**：比较随意、凌乱的图形可以考虑。
    5.  **封闭开放**：出现简笔画或图形凌乱时可以考虑。

### 二、定量分析 (点线面角素)
*   **口诀：点线面角素 —— 什么时候考虑定量分析？**
    1.  图形越来越复杂，数量在增加，点、线很多。
    2.  图形规整且复杂。
*   **考点：**
    1.  **点数量**：包括曲直交点、切点等。
    2.  **线数量**：包括直线、曲线、笔画数 (一笔画/多笔画)。
    3.  **面数量**：部分数、封闭空间数、面积。
    4.  **角数量**：直角、锐角、钝角。
    5.  **元素数量**：考察元素种类和某种元素数量。

### 三、位置分析 (直接想位移)
*   **口诀：直接想位移**
    1.  **垂直平行关系**：出现相似三角形、四边形时也要考虑平行线组数。
    2.  **连接方式**：点连接、线连接、连接处线段数。
    3.  **方向**：对称轴方向、线段端点方向等。
    4.  **位置**：内/外、黑/白点位置等。
    5.  **移动**：旋转、翻转、平移 (循环移动/往返移动)。

### 特殊考法
*   **元素换算**：题干中只有两种元素，数量在逐步增多时可考虑。(图2元素 $\times$ 2 = 图1元素 + 图3元素)
*   **叠加**：去同存异、去异存同、黑白运算，和“翻转”结合时较难。

## 三大类图形特征

### 1. 背景图 (三大哥、六小弟)
*   **移动、叠加、部分分数**。
*   **口诀：笔面连报是对滴** (笔画数、面积/周长、连接方式、抱团、对称、递推)。

### 2. 分割图 (连接、大小、三不沾、边数、对称、相似)
*   **口诀：连打三遍对象**。
*   **考查方式**：
    1.  连接方式：连接处线段数。
    2.  大小：图形中最大或者最小的部分。
    3.  三不沾：图中有一部分和外部轮廓不挨着。
    4.  边数：切割后图形很有规律。
    5.  对称：图中最大面最小面存在对称关系。
    6.  相似：存在相同图形。

### 3. 对称图 (轴对称、中心对称)
*   **三大号**：数量(对称轴数量)、方向(对称轴方向)。
*   **四小弟**：对称轴与原图形点/线/面的关系、位置关系(两条对称轴的位置关系)。

---

## 六大提示

1.  **数量加减**：规整的外部轮廓，多为多边形 $\rightarrow$ 多考察外部线条与内部点、线、面之和差。
2.  **笔画数**：
    *   **一笔画判定**：
        *   ① 奇点数 = 0 或 2
        *   ② 连在一起的一部分图形
    *   *注：点出出的线条数为奇数，则是奇点。*
    *   **多笔画**：
        *   ① 多部分图形 (多部分的笔画数相加)
        *   ② 奇点数 > 2，笔画数 = 奇点/2。
3.  **平行组数**：提示点为两个相同或相似三角形。
4.  **直角数**：提示点为垂线、电话卡(修正图形)、直角数。
5.  **圆提示**：有圆有线 (相切、相交、相离)；圆内有线 (圆内交点、圆内空间、圆上交点)。
6.  **连接方式**：提示点为简笔画、风车、树叶等。

## 立体图形

### 1. 立体展开图
*   **第一步：还原母图** (标号法 1-6)。
*   **第二步：三板斧**
    *   **相对面**：13/24/56是对立面不可能同时出现。
    *   **14可以直接平移四格不变**。
    *   **旋转**：移动一格相当于旋转90度。
*   **特殊展开图**：原图不重要，重点要关注选项，盯住公共边。

### 2. 立体截面图
*   **长方体**：四边形、三角形、梯形、六边形、正方形。
*   **圆柱体**：圆、椭圆、矩形、酒桶形/半椭圆。
*   **圆锥体**：圆形、三角形、椭圆、半椭圆。
*   *注意：截面图一定要注意有没有实线，有坑！*

# 10. 秒杀及补充

## 秒杀口诀

1.  **“两个有的”**
    *   **找点名那句，反着说，人称变所有。**
    *   例：(1)有的同学是党员 (2)有的同学不是党员 (3)涛涛是党员
    *   【解】点名那句是(3)，反着说：涛涛不是党员。人称变“所有”：所有人都不是党员。

2.  **“两个所有”**
    *   **找点名那句，顺着说，人称变所有。**
    *   例：(1)所有同学是党员 (2)所有同学不是党员 (3)涛涛和龙龙是党员
    *   【解】点名那句是(3)，顺着说：涛涛和龙龙是党员。人称变“所有”：所有人都是党员。

3.  **“两个有的，一个所有” (三段论)**
    *   **去同存异，前在前，后在后。**
    *   例：有些南京人不爱吃辣椒，那么有些爱吃甜食的南京人肯定不爱吃辣椒。
    *   【解】有些[南京人]不爱吃辣椒，有些[爱吃甜食的人][不爱吃辣椒] $\rightarrow$ [南京人]爱吃甜食。

4.  **“一人全对，一人全错，一人一对一错”**
    *   **找出现次数最多，为答案。**
    *   例：甲乙丙看车... “奇瑞”出现次数最多，因此这辆车是“奇瑞”。

5.  **“每个人都只猜对了一半”**
    *   **找出现次数最少，为答案。**
    *   例：小唐、小王、小赵去工作... “小赵”出现次数最少，即“小赵去甲公司工作”为真。
    *   *注：当选项中有两个均提到正确部分，可利用题干错误部分进行排除。*

6.  **“只有 xx 说真话”**
    *   **找 xx 这句话，说的话和提到的都为假。**
    *   例：只有会计师的预言是正确的... 提到“会计师”，即“乙：丙会成为会计师”是假的，推出“丙不会成为会计师”。

7.  **“只有一人说真话”**
    *   **找矛盾 (一真一假)，绕开矛盾 (其余均为假)，回到矛盾 (根据推断找答案)。**

8.  **“3+2”**
    *   **找相同，相同归 3，做排除；不同条件中，出现两次的被选中。**
    *   例：3人为博士... 张三、王五性别相同... 王五和赵六学历不同...
    *   【解】找相同：“张三、王五性别相同” “李四与钱七的学历相同”，根据“相同归3” —— 3人为博士，3人为男性。即张三、王五性别为男。

9.  **“4+3+2+1”**
    *   **用 3 找不同的人，用 2 找出不同人的队友，队友为答案。**
    *   例：3人责任心强，2人能力突出...
    *   【解】用3找不同的人：责任心强 $\rightarrow$ 乙、丙都有责任心；丙、丁并非都有责任心 $\rightarrow$ 丁没有责任心 $\rightarrow$ **不同人：丁**。用2找出不同人的队友：管理能力突出 $\rightarrow$ 甲、乙管理水平相当 $\rightarrow$ 丙、丁为一组 $\rightarrow$ **丙是丁的队友** $\rightarrow$ 选丙。

10. **“大小比较”**
    *   **最大信息排除比大小。**
    *   列出信息出现次数，排不干净的，考虑最大信息法。

11. **“数据拆分比大小”**
    *   **大大则大。**
    *   A县大学生村官中，本科学历 > 研究生，女性 > 男性。
    *   【解】本科学历+女性 > 研究生+男性。

## 常用逻辑关系 40 组 (部分)

| 序号 | 原命题 | 表达式 |
| :--- | :--- | :--- |
| 1 | 如果(只要) A，那么(就) B | $A \rightarrow B$ |
| 2 | 只有(除非) A，才(否则不) B | $B \rightarrow A$ |
| 3 | 除非(必须) A，否则 B | $-B \rightarrow A$ |
| 4 | 所有 A 都是 B | $A \rightarrow B$ |
| 5 | 所有 A 都不是 B | $A \rightarrow -B$ |
| 6 | 有的 A 是 B | 有的 $A \rightarrow B$ |
| 10 | A 当且仅当 B | $A \leftrightarrow B$ |
| 11 | 要么 A，要么 B | A或B 且 -(A且B) |
| 14 | 虽然 A，但是 B | A 且 B |
| 23 | 所有 A 都是 B | 并非有的 A 不是 B |
| 28 | $A \rightarrow B$ | -A 或 B |
| 29 | -(A 且 B) | -A 或 -B |
| 39 | $A \rightarrow -A$ | “-A” 为真 |
| 40 | $-A \rightarrow A$ | “A” 为真 |
