## 一、和差倍比与方程法

[例题](./例题/一.pdf)

> 找最小公倍数设同一个未知数 



### 普通方程
    
1.   **找等量关系 (和差倍比)：**
    *   设中间量： 各个量通过运算得到相同的值
    *   设小不设大： 避免分数
    *   设未知数：求谁设谁，*如果问的是倍数关系，将分母设为 1* 

2.   **代入排除法：**
    *   先排除： 利用选项尾数、奇偶、倍数等特性
    *   再代入： 最值原则、好算原则
    *   *蒙题技巧：* 选项三奇一偶选其偶，选项三偶一奇选其奇；问题小往注选第二小，问最大往注选第二大

### 不定方程

1.   **未知数的个数大于方程的个数**
    *   **奇偶性：** 未知数前的系数存在 **一奇一偶**
    *   **尾数性：** 未知数前的系数的尾数是 **0或5**
    *   **倍数法：** 未知数的系数与常数存在整除
    *   **代入法：** 将选项代入到题目的等量关系中

2.  不定方程组
    *  **加减消元法**
    *  特值法：未知数有小数，设某个未知数为0 (最难算的赋0)

## 二、工程问题

[例题](./例题/二.pdf)

*   核心公式：**工作量 = 效率 × 时间**
*   拓展公式：**工作总量 = 效率和 × 时间 (常用干合作完工问题)**
*   解题本质：**求效率，推算工作总量**



### 已知完工时间 
> 甲需要多少时间，乙需要多少时间，甲乙合作需要多少时间...

- **设最小公倍数为工作总量，再求效率**

### 已知效率比例
- **将效率比值直接当作效率，推算工作总量**

### 已知不同完工情况
> 甲先做多少天，乙做多少天，能够完成多少...

- **列方程求效率比 & 工作总量**

## 三、整除问题和合作完工问题

[例题](./例题/三.pdf)

### 利用倍数特性解不定方程
  *   ...占比... ,...占比... 
  *   $y=\frac{A}{B}x$ → **x 是 B 的倍数，y 是 A 的倍数** 
### 整除特性
*  若 a、b 能被 c 整除，则 **a+b、a-b 也能被 c 整除**
*  若一个数能被 *3、9* 整除，则该数的 **各位数字和也能被 3、9 整除**
*  若一个数能被 *2 / 4 / 8* 整除，则该数的 **末位/末二位/末三位 也能被 2 / 4 / 8 整除**
*  若一个数能被 *5 / 25 / 125* 整除，则该数的 **末位/末二位/末三位 也能被 5 / 25 / 125 整除**
*  多个数相乘看因子
> 22 × 23 × 24 *(3 * 8)* → 能被 3,8... 整除
>
> 24 × 27 *(3 × 9)* → 能被 3,9,...整除
### 合作完工问题
*   中途有人休息，看工作的人
*   中途没人休息，只是换地方干，则：$\frac{总工作量}{总效率}=总时间$

## 四、余数、平方数与等差数列

[例题](./例题/四.pdf)

### 数列基础知识
> 一般考等差数列，注意奇数项找 **中间项**

**数列常考公式：**

| | 通项公式 | 求和公式 |
| :--- | :--- | :--- |
| **等差数列** | $a_n = a_1 + (n - 1)d$ | $S_n = \frac{n(a_1 + a_n)}{2}$ |
| **等比数列** | $a_n = a_1 \times q^{n-1}$ | $S_n = \frac{a_1(1 - q^n)}{1 - q}$ |

**数列常见性质：**

| | 性质 |
| :--- | :--- |
| **等差数列** | 一、若是奇数项等差数列，则 $平均数=等差中项，S_n=n×等差中项；$ <br> $等差数列的平均数=\frac{a_1+a_n}{2}$；<br>二、若 m+n=k+i，则 $a_m + a_n = a_k + a_i$。 |
| **等比数列** | 一、若 $a_{n-1}, a_n, a_{n+1}$ 成等比数列，则 $a_n^2 = a_{n-1} a_{n+1}$；<br>二、若 m+n=k+i，则 $a_m a_n = a_k a_i$。 |



### 平方数问题

常见平方数

*   $11^2=121$, $12^2=144$, $13^2=169$, $14^2=196$, $15^2=225$
*   $16^2=256$, $17^2=289$, $18^2=324$, $19^2=361$


### 余数问题
> **余同加余，和同加和，差同减差**

| 特征 | 余同<br>余数相同 | 和同<br>除数与余数的和相同 | 差同<br>除数与余数的差相同 |
| :--- | :--- | :--- | :--- |
| **口诀** | 余同取余 | 和同加和 | 差同减差 |
| **被除数** | 最小公倍数+**余数** | 最小公倍数+**和** | 最小公倍数-**差** |
| **实例** | “一个数<br>除3余1，<br>除5余1，<br>除6余1”<br>则被除数为 30n+1 | “一个数<br>除7余1，<br>除6余2，<br>除5余3”<br>则被除数为 210n+8 | “一个数<br>除7余5，<br>除6余4，<br>除3余1”<br>则被除数为 42n-2 |

## 六、溶液问题与十字交叉法

[例题](./例题/六.pdf)

### 溶液问题
*   公式：
    *   浓度 = 溶质 ÷ 溶液
    *   溶液 = 溶质 + 溶剂
*   纯浓度问题(如：盐水、酒精、xx 溶液...)
    *   抓住 **溶质不变**：无论溶液如何改变，溶质质量不会凭空发生改变


### 十字交叉法
!!! info ""
    溶液A浓度 R，混合溶液浓度，溶液B浓度 R

    $\frac{R-B}{A-R} = \frac{\text{溶液A质量}}{\text{溶液B质量}}$

*   类浓度问题（如：平均分、平均 xx、增长率...）
    *   $A=\frac{B}{C}$ → **代入 A 求出来的是 C 的比**
    *   *求什么就把什么放到分母*
*   混合思想：部分 1 浓度 < 整体浓度 < 部分 2 浓度

## 七、经济利润问题

[例题](./例题/七.pdf)

### 基础题型

*   常用公式： 
    *   售价 = 成本(即进价) + 利润
    *   利润率 = 利润/收入
    *   总成本/总利润/总销售金额 = 单个成本/单个利润/单个售价 × 数量
    *   【注意】售价 $\neq$ 定价，售价是最终出售的，定价是最初的
*   解题方法：*找等式关系列方程*
*   其他名词
    *   折扣：指实际售价为原定价的几成，三折即原定价的30% (折扣和成本无关，和原定价有关)
        *   【注意】*折扣率指的是折扣为原定价的几成，与折扣正相反，若打三折，则折扣率为70%*
    *   利润率：指的是利润占成本的比例，若成本为100，利润为30，则利润率为30%  


### 阶梯计费
> 分段计费，可列方程解（也可画线段辅助）


### 分批销售
> 先按照某个价格售卖多少，再按照某个价格售卖多少...


*   设未知数：问什么设什么（若问 A 是 B 的多少倍，可设 B 为 1）
*   解题方法：用每一批的收入列方程（较为固定）
    *   公式：$第一批收入 + 第二批收入 + ... = 总收入 = 总成本 + 总利润$
